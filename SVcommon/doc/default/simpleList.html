Class Details

Clase para presentar los listados en la página.


Estan los listados tanto para artículos como para productos y se pueden usar combinados o separados
Cada listado tiene parametros que lo configuran, sin embargo los parametros de la clase, tanto los propios como los heredados de widget aplican para todos los métodos, según el caso en algunos pueden tener o no significado

articulosDestacados

void articulosDestacados( )

[line 168]

Permite listar los articulos destacados del site. Utiliza las siguientes variables


titulo, Si no esta configurado toma el valor por defecto "Artículos Destacados", si se coloca como valor "NO", no se carga ningun valor
autorArticulo, Si no esta configurado toma el valor por defecto "1"
rutaBase, permite configurar la ruta desde donde se toman las imagenes
formatoPresentacion, ubicación del archivo de presentación de los articulos encontrados
formatoTitulo, ubicación del archivo de presentación del titulo de los articulos. Si se coloca como valor "NO", no se muestra el formato

  1.  <?php
  2.  $widget = new simpleList();
  3.  $widget->formatoPresentacion="SVwidgets/SimpleList/formato-articulo.php";
  4.  $widget->formatoTitulo="SVwidgets/SimpleList/formato-titulo.php";
  5.  $widget->titulo="Importante...";
  6.  $widget->rutaBase = $DOMINIOSV;
  7.  $widget->articulosDestacados();
  8.  ?>

Permite mostrar los artículos dadas unas condiciones específicas
Su uso no se recomienda, ha sido creado para ser usado por métodos de la clase que enmascaren los querys.

Parameters:

  • String $titulo, - titulo del widget a ser mostrado en el site
  • String $condiciones, - condiciones del query para la base de datos
  • String $orderBy, - campos de la base de datos por los cuales se va a ordenar el listado
  • Number $maxResults, - número máximo de resultados que se van a presentar, si no se indica el valor se asume valor por defecto = 0 (es decir, muestra todos los resultados que se obtengan)
