method articulosDestacados [line 170]

void articulosDestacados( )

Permite listar los articulos destacados del site. Utiliza las siguientes variables


titulo, Si no esta configurado toma el valor por defecto "Artículos Destacados", si se coloca como valor "NO", no se carga ningun valor
autorArticulo, Si no esta configurado toma el valor por defecto "1"
rutaBase, permite configurar la ruta desde donde se toman las imagenes
formatoPresentacion, ubicación del archivo de presentación de los articulos encontrados
formatoTitulo, ubicación del archivo de presentación del titulo de los articulos. Si se coloca como valor "NO", no se muestra el formato

  1.  <?php
  2.  $widget = new simpleList();
  3.  $widget->formatoPresentacion="SVwidgets/SimpleList/formato-articulo.php";
  4.  $widget->formatoTitulo="SVwidgets/SimpleList/formato-titulo.php";
  5.  $widget->titulo="Importante...";
  6.  $widget->rutaBase = $DOMINIOSV;
  7.  $widget->articulosDestacados();
  8.  ?>

method categoriasDeArticulos [line 272]

void categoriasDeArticulos( [Number $nivel = 1], [boolean $mostrarArticulos = true], [boolean $incluirGeneral = true])

Permite encontrar las categorias y subcategorias de articulos que están configuradas.

Según las configuraciones se muestran mas o menos niveles de categorias y los articulos de la misma. Además de los parámetros toma en cuenta lo siguientes parametros de la clase formatoPresentacion, formato para las categorías. Se debe colocar en el formato el tag <element/> para indicar donde se quieren los hijos. formatoPresentacionHoja, formato del último nivel del widget. Si se configura para mostrarse 4 niveles, pero en el nivel 2 alguna categoría no tiene mas hijos, los articulos (si están configurados para salir) se muestran con este formato formatoTitulo, formato del titulo del widget rutaBase, próximo a ser eliminado, indica la ruta base de las imagenes. Se configura con $DOMINIOSV usualmente En el comportamiento estandar de la función se incluye la revisión de la permisología de las categorías. Si en la sesión no hay usuario logueado solo se muestran las categorías con only_for=0 Si en la sesión hay un usuario logueado se muestran las categorias con only_for=0,1 y cualquier grupo al cual el pertenezca.




Tags:

example:  Agregar las categorias a una barra


Parameters:

Number   $nivel   Determina el nivel de categorias que se muestran
boolean   $mostrarArticulos   Indica si se muestran los articulos en el nivel final. Acepta los valores true para mostrarlos false para no mostrarlos
boolean   $incluirGeneral   La categoria #1
